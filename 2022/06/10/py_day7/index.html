Python学习7-字典

发表于2022-06-10|更新于2022-06-10
|阅读量:

1、使用字典

1.1、字典是一系列的k-v对。
1.2、用法：student = {‘name’: ‘zhouyu’, ‘age’: 25, ‘sex’: ‘man’}
空字典： student = {}
1.3、增：
student[‘weight’] = 65
student[‘nation’] = ‘china’

1.4、删：
del student[‘weight’]

1.5、改：
student[‘weight’] = student[‘weight’] + 10
student[‘name’] = ‘zhangyifei’

2、遍历字典

2.1、遍历k-v对：
for k, v in student.items():
print(k)
print(v)

2.2、遍历key：
for k in student.keys():
print(k)

for k in student:
print(k)

2.3、按顺序遍历key：
for k in sorted(student.keys()):
print(k)

2.4、遍历value：
for v in student.values():
print(v)

2.5、用集合set剔除重复：
for v in set(student.values()):
print(v)

3、嵌套

3.1、列表中嵌套字典：
student1 = {‘name’: ‘zhouyu’, ‘age’: 25, ‘sex’: ‘man’}
student2 = {‘name’: ‘zhangyifei’, ‘age’: 26, ‘sex’: ‘man’}
student3 = {‘name’: ‘wanghaozun’, ‘age’: 24, ‘sex’: ‘man’}

students = [student1, student2, student3]

for stu in students:
print(stu)

3.2、字典中嵌套列表（一个key对多个value）：
students = {
‘zhouyu’: [25, ‘man’],
‘zhangyifei’: [26, ‘man’],
‘wanghaozun’: [24, ‘man’]
}
for name, ageAndSex in students.items():
print(name)
print(ageAndSex)

3.3、字典中嵌套字典
school = {
‘students’: {
‘zhouyu’: [25, ‘man’],
‘zhangyifei’: [26, ‘man’],
‘wanghaozun’: [24, ‘man’]
},
‘teachers’: {
‘lishumao’: [45, ‘man’],
‘liujing’: [30, ‘woman’],
‘wangyingchun’: [65, ‘man’]
}
}
for stu, teacher in school.items():
print(stu)
print(teacher)
